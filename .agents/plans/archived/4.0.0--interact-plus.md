# Embed Interaction UX Improvements

## Problem Statement

Users are confused about how to interact with embedded content (PrototypeEmbed, FigmaEmbed, ComponentWidget). The current double-click interaction is not discoverable — users don't know they need to double-click to make the embed interactive.

## Solution

1. **Add hover overlay with "Click to interact" message** — When hovering over a non-interactive embed, show a semi-dark overlay with a centered "Click to interact" label
2. **Change from double-click to single-click** — Reduce friction by making embeds interactive with a single click (which also selects the widget)

## Files to Change

1. **PrototypeEmbed.jsx** — Change `onDoubleClick` to `onClick` on dragOverlay, add hover hint markup
2. **PrototypeEmbed.module.css** — Add styles for hover overlay with centered message
3. **FigmaEmbed.jsx** — Same changes as PrototypeEmbed
4. **FigmaEmbed.module.css** — Same CSS changes
5. **ComponentWidget.jsx** — Same changes
6. **ComponentWidget.module.css** — Same CSS changes

## Implementation Steps

### Step 1: Create shared overlay styles
Each embed already has a `.dragOverlay` or `.interactOverlay` class. Add:
- Dark background on hover (rgba(0,0,0,0.4) or similar)
- "Click to interact" text centered with flexbox
- Text styling: white, semi-bold, with subtle text-shadow

### Step 2: Update PrototypeEmbed
- Change `onDoubleClick={enterInteractive}` to `onClick={enterInteractive}`
- Add the hint text inside the overlay div

### Step 3: Update FigmaEmbed
- Same changes as PrototypeEmbed

### Step 4: Update ComponentWidget
- Same changes as PrototypeEmbed

## Edge Cases & Risks

- **Selection behavior**: Single click should still allow widget selection by WidgetChrome — need to ensure `onClick` doesn't interfere. The overlay `onClick` should trigger interactive mode AND the click should bubble up to select the widget.
- **Mobile/touch**: Single tap should work consistently
- **Accessibility**: The overlay should be focusable and keyboard-operable (Enter to interact)
